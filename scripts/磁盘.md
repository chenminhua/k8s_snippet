## azure

创建磁盘
az disk create -g test -n tv1-data --size-gb 100

挂载磁盘到机器
az vm disk attach -g test --vm-name tv1 --disk tv1-data

ls -l /dev/sd*


detach一块磁盘
az vm disk detach -g test --vm-name tv1 -n tv1-data-dev

删除磁盘
az disk delete -g test -n tv1-data
